# 快速安装指南

## 📋 安装前准备

### 1. 生成图标文件

打开 `icon-generator.html` 文件（双击或在浏览器中打开）：

```bash
# Windows
start icon-generator.html

# Mac
open icon-generator.html

# Linux
xdg-open icon-generator.html
```

点击「下载全部图标」按钮，将下载的4个PNG文件保存到 `icons/` 目录下。

## 🚀 安装到Chrome

### 步骤1：打开扩展程序页面

在Chrome浏览器地址栏输入：
```
chrome://extensions/
```

### 步骤2：启用开发者模式

在页面右上角打开「开发者模式」开关。

### 步骤3：加载扩展程序

1. 点击左上角「加载已解压的扩展程序」按钮
2. 选择本项目的 `gemini-summarizer` 文件夹
3. 点击「选择文件夹」

### 步骤4：验证安装

✅ 在扩展程序列表中看到「Gemini页面总结助手」
✅ 浏览器工具栏出现插件图标
✅ 没有错误提示

## 🎯 开始使用

### 测试插件

1. 打开任意网页（例如：新闻文章、博客等）
2. 点击浏览器工具栏的插件图标
3. 在弹出窗口点击「发送到Gemini总结」
4. 等待Gemini页面打开并自动填充内容

### 设置快捷键

1. 访问 `chrome://extensions/shortcuts`
2. 找到「Gemini页面总结助手」
3. 点击输入框，按下你想要的快捷键（例如：Ctrl+Shift+G）
4. 保存

现在可以使用快捷键快速触发插件！

## ⚙️ 配置选项

点击插件图标，在弹出窗口中可以配置：

- **后台打开Gemini页面**：默认勾选，Gemini在后台标签页打开
- **快捷键设置**：点击「修改快捷键」跳转到快捷键设置页面

## 🐛 常见问题

### 问题1：找不到图标文件

**解决方法**：
1. 确保已使用 `icon-generator.html` 生成图标
2. 检查 `icons/` 目录下是否有这4个文件：
   - icon16.png
   - icon32.png
   - icon48.png
   - icon128.png

### 问题2：插件加载失败

**解决方法**：
1. 检查是否启用了「开发者模式」
2. 确保选择了正确的文件夹（包含 manifest.json）
3. 查看错误提示信息，检查缺失的文件

### 问题3：点击插件没反应

**解决方法**：
1. 打开浏览器控制台（F12）查看错误信息
2. 在 `chrome://extensions/` 页面查看插件错误
3. 尝试重新加载插件（点击刷新图标）

### 问题4：Gemini页面未自动填充内容

**可能原因**：
- Gemini页面结构已更新
- 页面加载时间较长
- 网络连接问题

**解决方法**：
1. 等待Gemini页面完全加载
2. 手动将内容复制粘贴到输入框
3. 检查是否已登录Google账号

## 📦 更新插件

当代码有更新时：

1. 访问 `chrome://extensions/`
2. 找到「Gemini页面总结助手」
3. 点击刷新图标（🔄）
4. 插件会重新加载最新代码

## 🗑️ 卸载插件

1. 访问 `chrome://extensions/`
2. 找到「Gemini页面总结助手」
3. 点击「移除」按钮
4. 确认卸载
