## 📥 安装

### Chrome Web Store（即将上线）
扩展正在审核中，很快就能在 Chrome Web Store 下载安装。

### 手动安装
1. 下载 `gemini-summarizer-v1.3.0.zip`
2. 解压到本地目录
3. 打开 Chrome 浏览器，访问 `chrome://extensions/`
4. 开启「开发者模式」
5. 点击「加载已解压的扩展程序」
6. 选择解压后的目录

## 🔧 使用方法

1. **配置 API Key**
   - 点击扩展图标，进入配置页面
   - 输入您的 Gemini API Key
   - （可选）自定义 Prompt

2. **使用扩展**
   - 浏览任意网页
   - 点击扩展图标或使用快捷键
   - 内容将自动发送到 Gemini 进行分析

## 📋 系统要求

- Chrome 浏览器 88 及以上版本
- Google Gemini API Key（免费获取：https://makersuite.google.com/app/apikey）
