# 🚀 快速开始指南

## 📦 安装步骤（3分钟搞定）

### 1️⃣ 生成图标（必需）

1. 双击打开 `icon-generator.html` 文件
2. 点击页面上的「📦 下载全部图标」按钮
3. 将下载的4个PNG文件保存到 `icons/` 文件夹中
   - icon16.png
   - icon32.png
   - icon48.png
   - icon128.png

### 2️⃣ 安装到Chrome

1. 打开Chrome浏览器，地址栏输入：`chrome://extensions/`
2. 右上角打开「**开发者模式**」开关
3. 点击「**加载已解压的扩展程序**」
4. 选择 `gemini-summarizer` 文件夹
5. 完成！🎉

### 3️⃣ 设置快捷键（可选）

1. 访问：`chrome://extensions/shortcuts`
2. 找到「Gemini页面总结助手」
3. 点击输入框，按下快捷键（推荐：`Ctrl+Shift+G`）

## 🎮 使用方法

### 方式1：点击图标

```
1. 打开任意网页（如新闻、文章、博客）

```

## ❓ 常见问题

### Q1: 插件图标不显示？

**A**: 检查 `icons/` 目录下是否有4个PNG文件。如果没有，请使用 `icon-generator.html` 生成。

### Q2: 点击后没反应？

**A**:
- 按F12打开浏览器控制台，查看错误信息
- 在 `chrome://extensions/` 检查插件是否启用
- 尝试刷新页面后再试

### Q3: 内容提取不完整？

**A**:
- 某些动态加载的网站可能需要滚动后再提取
- 部分网站有反爬虫限制
- 确保页面完全加载完成

### Q4: Gemini页面没有自动填充？

**A**:
- 等待Gemini页面完全加载（约2-3秒）
- 如未自动填充，手动复制粘贴到输入框
- 检查是否已登录Google账号

### Q5: 可以在所有网站使用吗？

**A**: 大部分网站都支持，但以下情况可能受限：
- 需要登录的内容页面
- 使用特殊反爬虫技术的网站
- PDF文件（需先转换为网页）
- 视频内容（仅提取文字和截图）

## 🔧 故障排查

### 情况1：加载失败

```bash
# 检查文件完整性
cd gemini-summarizer
ls -la

# 确认必需文件都存在：
# ✓ manifest.json
# ✓ background.js
# ✓ content-extractor.js
# ✓ popup.html/css/js
# ✓ icons/*.png
```

### 情况2：功能异常

1. 在 `chrome://extensions/` 找到插件
2. 点击「错误」按钮查看日志
3. 点击刷新图标（🔄）重新加载
4. 如果问题持续，尝试卸载后重新安装

### 情况3：权限问题

如果Chrome提示权限警告：
- 这是正常现象
- 插件需要这些权限才能提取网页内容
- 所有权限仅在本地使用，不会上传数据
- 详见 `PRIVACY.md` 了解隐私保护措施
