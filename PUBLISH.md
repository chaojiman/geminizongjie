## 📦 打包发布

### 步骤1：最终检查

```bash
# 确保在项目目录
cd gemini-summarizer

# 检查所有文件
ls -la

# 确认以下文件存在：
# ✓ manifest.json
# ✓ background.js
# ✓ content-extractor.js
# ✓ popup.html
# ✓ popup.css
# ✓ popup.js
# ✓ icons/icon16.png
# ✓ icons/icon32.png
# ✓ icons/icon48.png
# ✓ icons/icon128.png
# ✓ README.md (可选，但推荐)
```

### 步骤2：创建ZIP包

#### Windows (PowerShell)
```powershell
Compress-Archive -Path * -DestinationPath gemini-summarizer.zip -Force
```

#### Mac/Linux
```bash
zip -r gemini-summarizer.zip * -x "*.git*" "*.DS_Store" "node_modules/*"
```

### 步骤3：上传到Chrome Web Store

1. 访问 [Chrome Web Store 开发者控制台](https://chrome.google.com/webstore/devconsole/)
2. 登录Google账号
3. 点击「新增项」
4. 上传ZIP文件
5. 填写商店信息
6. 提交审核

## 📊 审核时间线

- **提交后**: 立即收到确认邮件
- **审核中**: 1-3个工作日
- **审核通过**: 收到邮件通知，插件上架
- **审核拒绝**: 收到拒绝原因，修改后重新提交

## 🚫 常见拒绝原因

1. **权限过度**: 申请了不必要的权限
2. **功能不符**: 实际功能与描述不符
3. **隐私问题**: 未说明数据收集和使用
4. **商标侵权**: 名称或图标侵犯他人商标
5. **质量问题**: 功能不完整或有明显bug
6. **远程代码**: 从外部加载脚本（违反政策）
