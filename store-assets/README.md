# Chrome Web Store 素材准备指南

## 必需素材

### 1. 小型宣传图（必需）
- **文件名**: `promo-small.png`
- **尺寸**: 440x280 像素
- **用途**: Chrome Web Store 展示用
- **要求**: 
  - 清晰展示插件核心功能
  - 包含插件名称或主要功能说明
  - 使用高质量设计

### 2. 功能截图（至少3张，建议5张）
- **文件名**: `screenshot1.png`, `screenshot2.png`, `screenshot3.png` 等
- **尺寸**: 1280x800 或 640x400 像素
- **要求**: 
  - 展示真实功能，不能虚假宣传
  - 可以添加文字说明突出功能点

#### 建议的截图内容：

**截图1 - 插件主界面**
- 展示 popup 窗口
- 显示主要功能按钮
- 展示设置选项

**截图2 - 内容提取过程**
- 展示正在提取网页内容的状态
- 显示提取进度提示

**截图3 - Gemini自动填充效果**
- 展示 Gemini 页面自动填充内容的效果
- 显示自动提交功能

**截图4 - 配置管理界面**
- 展示配置管理页面
- 显示提示词模板管理功能

**截图5 - 使用场景示例**
- 展示在不同类型网站上的使用效果
- 显示分析结果示例

## 可选素材

### 大型宣传图
- **尺寸**: 920x680 或 1400x560 像素
- **用途**: 增强展示效果

### 宣传视频
- **格式**: YouTube 链接
- **时长**: 60秒以内
- **内容**: 演示插件使用流程

## 制作建议

1. **使用专业工具**: 
   - Figma、Adobe Illustrator 设计宣传图
   - 使用截图工具（如 Snagit、ShareX）制作功能截图

2. **保持一致性**:
   - 使用统一的配色方案
   - 保持设计风格一致

3. **突出核心功能**:
   - 清晰展示主要功能
   - 使用简洁的文字说明

4. **真实展示**:
   - 使用真实的界面截图
   - 避免过度美化或虚假宣传

## 文件结构

```
store-assets/
├── promo-small.png          # 小型宣传图（必需）
├── screenshot1.png          # 截图1 - 主界面
├── screenshot2.png          # 截图2 - 内容提取
├── screenshot3.png          # 截图3 - Gemini效果
├── screenshot4.png          # 截图4 - 配置管理
└── screenshot5.png          # 截图5 - 使用场景
```

## 注意事项

- 所有素材必须原创或使用有授权的资源
- 截图需要展示真实功能，不能虚假宣传
- 确保图片清晰，符合尺寸要求
- 避免使用受版权保护的图片或内容
